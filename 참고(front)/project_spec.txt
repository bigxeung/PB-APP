3. 필수 구현 내용 ⭐⭐⭐
================================================================================

3-1. Front-End Web Application
-------------------------------------------
다음 중 1개 이상 선택:
  [ ] React.js
  [ ] Vue.js
  [ ] Streamlit

주요 기능:
  - 사용자 인터페이스(UI) 구성
  - Firebase와 데이터 송수신(읽기/쓰기)
  - 로그인/로그아웃 UI

3-2. Mobile Application
-------------------------------------------
다음 중 1개 필수 선택:
  [ ] React Native
  [ ] Flutter

⚠️ PWA(Progressive Web App)는 인정 안됨!
⚠️ WebApp과 동일한 Firebase 프로젝트 사용

3-3. Firebase 연동 ⭐⭐⭐
-------------------------------------------
필수 구성 요소:

1. Firebase Authentication (소셜 로그인 포함)
   - Google, GitHub, Facebook 등 1종 이상

2. Firebase Database (둘 중 선택)
   - Realtime Database 또는
   - Firestore

3. 배포
   ⚠️ WebApp 배포는 JCloud에 해야 함 (Firebase Hosting X)

중요: Web + Mobile 모두 같은 Firebase 프로젝트 연동!

5. 주의사항 & 가이드
================================================================================

보안:
- Firebase 설정 파일은 깃허브 업로드 가능
- API Key는 private key가 아님 (공개 가능)
- ⚠️ 개인정보, 비밀번호는 업로드 금지!

개발 방향:
- 너무 복잡한 기능보다 → 명확하고 완성도 있는 기본 기능에 집중
- WebApp + Mobile App이 동일한 데이터베이스 공유하는지 확인
- 어려우면 반드시 질문!
